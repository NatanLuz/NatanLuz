## Olá, eu sou o Natan

Sou Desenvolvedor Backend com foco em PHP e Laravel, construindo aplicações orientadas a regras de negócio, APIs REST e modelagem de banco de dados.

Meu objetivo profissional é desenvolver software bem estruturado, seguro, escalável e fácil de manter. Tenho atenção especial a arquitetura, organização de código, validação de dados, autorização e boas práticas de segurança.

Atualmente sigo evoluindo em arquitetura de software, testes automatizados e técnicas que elevam a qualidade de sistemas backend em produção.
